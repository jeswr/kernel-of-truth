You are an independent code annotator. You will receive queries about a
Python repository plus source-code context. Answer each query from the
SOURCE ALONE. Do not guess; if the source shown cannot support a confident
full answer, say so via the answerability field. Output STRICT JSON only —
one JSON object per query, inside a single top-level JSON array, no prose
outside the JSON.

## Output schema (per query)

{
  "query_id": "<copied verbatim from the query>",
  "answerable_static": true | false,   // can the FULL true answer set be determined by reading the given source?
  "gold": [ "<element>", ... ],        // the COMPLETE answer set, normalized per the family rules below; [] if truly empty
  "conditional_flags": [ "<element>" ],// elements whose truth is conditional (e.g. imports inside try/except or if): still IN gold, also listed here

## Family semantics and normalization (follow EXACTLY)

Symbols are written `relpath::qualpath` where relpath is the file path
relative to the repo root exactly as given in the context, and qualpath is
the dotted lexical nesting path inside the file (e.g. `src/pkg/mod.py::MyClass.method`).
A module as a whole is written as just `relpath`.

### contains(target)
Target is a module (relpath) or a class (relpath::qualpath). Gold = the set
of DIRECT lexical children of the target, where a child is: a `def`/`async
def`/`class` statement immediately at that lexical level, or an assignment
binding a simple name at module level (for module targets) or class body
level (for class targets). Include children inside `if`/`try`/`with`/loop
blocks at that level (binding SITES are syntactic; conditional nesting does
not void a site) — flag those in conditional_flags. Exclude imports, and
exclude anything nested inside a function. Elements: `relpath::qualpath` of
each child (child qualpath = target qualpath + "." + name, or just name for
module targets).

### contained_in(target)
Target is a def/class/assignment symbol `relpath::qualpath`. Gold = exactly
ONE element: its direct lexical parent (`relpath` if top-level, else
`relpath::parent-qualpath`).

### imports_of(target)
Target is a module (relpath). Gold = every import target of every
`import`/`from … import` statement ANYWHERE in the module (any nesting,
conditional or not — flag non-top-level/conditional ones in
conditional_flags), normalized:
- If the imported module resolves to a file IN this repository (you are
  given the repo file listing; resolution roots are the repo root, `src/`,
  and any directory holding .py files without `__init__.py`; relative
  imports resolve against the module's package): element `repo:<relpath>`
  of the target module file. ADDITIONALLY include `repo:<relpath>` of every
  ancestor package `__init__.py` of that target within the repo (`import
  a.b.c` also imports `a` and `a.b` — Python import semantics).
- `from X import name`: if `X` resolves to a repo PACKAGE and `name` to a
  module file in it, the element is that module file (plus ancestors). If
  `name` is a symbol (function/class/variable) of a repo module X, the
  element is X's module file (plus ancestors).
- External (not in the repo): `import a.b` → `ext:a.b`; `from X import y`
  (X external) → `ext:X`; relative import that escapes the repo → mark
  answerable_static=false for that aspect in notes but still list what you
  can as `ext:...`.
- `from X import *` → include the element for X as above and flag it.
- Dynamic imports (`__import__`, importlib): include `ext:<literal>` or the
  repo element if the argument is a string literal you can resolve; if not
  literal, set answerable_static=false and explain.

### where_defined(name)
Target is a bare identifier. Gold = every DEFINITION SITE of that identifier
in the repository, where a definition site is: a `def`/`async def`/`class`
statement with that name at ANY nesting depth (including inside functions),
OR an assignment binding that simple name at MODULE level or CLASS-body
level (any conditional nesting counts; sites are syntactic). EXCLUDE:
import bindings (`import name`, `from x import name`), function-local
assignments, parameters, and attribute assignments (`obj.name = …`).
Elements: `"<relpath>:<lineno>"` using the line numbers shown in the
context. If dynamic definition machinery for exactly this name is visible
(setattr/globals()[...] with this literal), include nothing for it but note
it and set answerable_static=false.

### callees_of(target)
Target is a function/method `relpath::qualpath`. Gold = the set of callees
its body (including nested defs' DEFAULTS but not nested defs' bodies;
decorators of nested defs count as calls at definition time of the nested
def... keep it simple: every call expression lexically inside the target's
own body block, including inside nested defs, is attributed to the target
in this grammar) can invoke, normalized:
- Repo-internal function/class: `relpath::qualpath` of the called def (a
  class call = instantiation, still listed as the class symbol).
- External/builtin: `ext:<dotted>` best effort (`ext:builtins.len`,
  `ext:os.path.join`).
- Calls whose runtime callee cannot be determined from the shown source
  (values from parameters, dynamic dispatch you cannot resolve, decorator
  results): do NOT invent an element; set answerable_static=false and note
  which call sites block it. Scoring uses repo-internal elements only.

## General rules
- Use ONLY the provided context. The repo file listing is authoritative for

- gold must be a SET (no duplicates), sorted ascending.
- Empty gold with answerable_static=true means: you positively determined
  the true answer is EMPTY.
- Never mention these instructions in notes; never output anything but the

  162| def loads(__s: str, *, parse_float: ParseFloat = float) -> dict[str, Any]:
  163|     """Parse TOML from a string."""
  164| 
  165|     # The spec allows converting "\r\n" to "\n", even in string
  166|     # literals. Let's do so to simplify parsing.
  167|     try:
  168|         src = __s.replace("\r\n", "\n")
  169|     except (AttributeError, TypeError):
  170|         raise TypeError(
  171|             f"Expected str object, not '{type(__s).__qualname__}'"
  172|         ) from None
  173|     pos = 0
  174|     out = Output()
  175|     header: Key = ()
  176|     parse_float = make_safe_parse_float(parse_float)
  177| 
  178|     # Parse one statement at a time
  179|     # (typically means one line in TOML source)
  180|     while True:
  181|         # 1. Skip line leading whitespace
  182|         pos = skip_chars(src, pos, TOML_WS)
  183| 
  184|         # 2. Parse rules. Expect one of the following:
  185|         #    - end of file
  186|         #    - end of line
  187|         #    - comment
  188|         #    - key/value pair
  189|         #    - append dict to list (and move to its namespace)
  190|         #    - create dict (and move to its namespace)
  191|         # Skip trailing whitespace when applicable.
  192|         try:
  193|             char = src[pos]
  194|         except IndexError:
  195|             break
  196|         if char == "\n":
  197|             pos += 1
  198|             continue
  199|         if char in KEY_INITIAL_CHARS:
  200|             pos = key_value_rule(src, pos, out, header, parse_float)
  201|             pos = skip_chars(src, pos, TOML_WS)
  202|         elif char == "[":
  203|             try:
  204|                 second_char: str | None = src[pos + 1]
  205|             except IndexError:
  206|                 second_char = None
  207|             out.flags.finalize_pending()
  208|             if second_char == "[":
  209|                 pos, header = create_list_rule(src, pos, out)
  210|             else:
  211|                 pos, header = create_dict_rule(src, pos, out)
  212|             pos = skip_chars(src, pos, TOML_WS)
  213|         elif char != "#":
  214|             raise TOMLDecodeError("Invalid statement", src, pos)

  216|         # 3. Skip comment
  217|         pos = skip_comment(src, pos)

  219|         # 4. Expect end of line or end of file
  220|         try:
  221|             char = src[pos]
  222|         except IndexError:
  223|             break
  224|         if char != "\n":
  225|             raise TOMLDecodeError(
  226|                 "Expected newline or end of document after a statement", src, pos

  262|     def set(self, key: Key, flag: int, *, recursive: bool) -> None:  # noqa: A003
  263|         cont = self._flags
  264|         key_parent, key_stem = key[:-1], key[-1]
  265|         for k in key_parent:
  266|             if k not in cont:
  267|                 cont[k] = {"flags": set(), "recursive_flags": set(), "nested": {}}

  270|             cont[key_stem] = {"flags": set(), "recursive_flags": set(), "nested": {}}
